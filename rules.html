15 || 0 Rules:

This is a basic connect four game with 0's opting for red circles and 1's opting for yellow.

    Rules:
  • Players must alternate turns
  • Players must click on lowest avaliable square in row
  • If Player 1 gets a 0 nibble in binary (4 0s in a row) = WIN 4 PL1
  • If Player 2 gets a 15 nibble in binary (4 1s in a row) = WIN 4 PL2
  • HAPPY GAMING
